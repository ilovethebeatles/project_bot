# project_bot
Мой телеграм-бот может искать фильмы похожие на указанный пользователем, выдавать фильмы, в которых есть определенный актер, а также подбирать фильмы на тематику, введенную пользователем.
Чтобы запустить бот, надо скачать файлы в одну папку вставить токен в файл config.py в переменную my_token вместо 'TOKEN' и запустить файл а терминале командой python3 botitself.py.
